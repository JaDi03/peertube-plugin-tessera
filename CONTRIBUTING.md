# Contributing to Tessera PeerTube Plugin

Thank you for your interest in contributing!

## Development Flow

1. Clone the repository.
2. Ensure you are using the correct Node.js version (`nvm use`).
3. Run `npm install` to install dependencies.
4. Run `npm run build` to compile the TypeScript files.
5. Create a feature branch and submit a PR against `main`.

Please adhere to the 100% English code standard for all comments and variable names.
